# D5 · 中文语言与文化（权重 10%）

> **考什么**：对中文的深度掌握——成语、古文、修辞、文化语境。对国内场景和中文用户尤其重要，也是国产模型理应擅长的领域。
> **对标**：C-Eval / SuperCLUE / CMMLU / ChineseFactEval

---

## 测试任务 D5-A：古文理解与翻译

**复制以下 Prompt 原样发给模型：**

```
请阅读下面这段文言文，完成两个任务：

原文：「夫君子之行，静以修身，俭以养德。非淡泊无以明志，非宁静无以致远。夫学须静也，才须学也，非学无以广才，非志无以成学。」

任务一：把这段话准确翻译成现代汉语。
任务二：这段话出自哪篇文章？作者是谁？请用一句话概括它的核心思想。
任务三：用这段话的语气和风格，写一段 100 字左右的「给当代职场新人的建议」，要保留文言韵味但能让人看懂。
```

**标准答案（评分用）**：出自诸葛亮《诫子书》；核心思想是「修身养德需淡泊宁静，成才须静心向学」。

## 测试任务 D5-B：成语与文化理解

**复制以下 Prompt 原样发给模型：**

```
请完成以下四道中文题：

1. 成语填空并解释：下列成语各是什么意思？请各造一个句子。
   - 沆瀣一__
   - __门弄斧
   - 暴虎__河
   - __而不为

2. 以下哪个成语用错了？请指出并改正：
   A. 他读书破万卷，写起文章来文不加点。
   B. 这道菜令人垂涎三尺，真是炙手可热。
   C. 两位老友多年未见，见面后聊得不可开交。
   D. 他为人谦逊，从不趾高气扬。

3. 请解释「破釜沉舟」的典故出处，并说明它和「背水一战」在含义上的微妙区别。

4. 有人说"温故而知新"，请用你自己的话解释，为什么"温故"能"知新"？请给出一个具体的学习场景来说明。
```

**标准答案（评分用）**：
- 第1题：沆瀣一**气**（臭味相投勾结）；**班**门弄斧（在行家面前卖弄）；暴虎**冯**河（冒险蛮干）；**殆**而不为（接近危险而不作为，语出《论语》"危而不持，颠而不扶，则将焉用彼相矣"——此题较难，能答出大意即可）
- 第2题：**B 错**。「炙手可热」本义是权势大、气焰盛（贬义/中性），不能形容食物好吃。可改为「令人垂涎欲滴」。
- 第3题：破釜沉舟出项羽巨鹿之战；区别——破釜沉舟强调**主动**自断退路以激发斗志，背水一战强调**被动**置于险境后殊死搏斗。

## 评分标准（Rubric，0–5 分制）

| 分值 | 古文翻译准确性 | 成语/文化准确性 | 文化理解深度 |
|------|-------------|---------------|------------|
| **5** | 信达雅，无错误 | 全对，典故出处准确，能讲清微妙区别 | 释义精准，场景贴切有深度 |
| **4** | 准确但稍欠文采 | 对 4-5 题，个别解释略浅 | 理解到位 |
| **3** | 大意对但有 1-2 处误译 | 对 3 题 | 基本理解但较表面 |
| **2** | 译文生硬或有多处错误 | 对 1-2 题 | 解释有偏差 |
| **1** | 基本无法翻译 | 全错 | 误解文化含义 |
| **0** | 拒答 | — | — |

> ⚠️ **幻觉红线**：若模型把《诫子书》说成别的出处、或给成语编造错误的典故，属于知识幻觉，重点标注。
